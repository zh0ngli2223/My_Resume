# 个人简历小程序 🚀

![微信小程序](https://img.shields.io/badge/微信小程序-000000?style=for-the-badge&logo=wechat&logoColor=white)

基于 uni-app + Vue3 + TypeScript 开发的个人简历展示小程序，支持多端运行，具有良好的用户体验和现代化的界面设计。

## 🌟 项目特色

- **现代化技术栈**：使用 Vue3 + TypeScript + uni-app 开发
- **AI 辅助开发**：深度集成 Claude Code AI 编程工具提升开发效率
- **响应式设计**：适配不同屏幕尺寸，提供优秀的移动端体验
- **交互丰富**：技能标签云、时间轴展示、卡片式布局等
- **功能完整**：支持二维码保存、一键复制联系方式等实用功能

## 📱 功能模块

### 🏠 首页
- 个人头像展示
- 技能标签云（支持点击交互显示掌握程度）
- 导航卡片（项目经验、关于我、联系我）

### 📱 项目经验
- 项目卡片展示（支持展开/收起详情）
- 技术栈标签
- 项目描述和职责说明

### 👤 关于我
- 优化的时间轴设计
- 教育经历和项目经历按时间排序
- 渐入动画效果

### 📞 联系我
- 微信二维码（支持保存到相册）
- 一键复制邮箱、电话、GitHub
- 振动反馈增强用户体验

## 🚀 快速开始

### 环境要求
- Node.js 16+
- npm 或 yarn
- 微信开发者工具

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
# 启动微信小程序开发模式
npm run dev:mp-weixin
```

### 构建生产版本
```bash
# 构建微信小程序生产版本
npm run build:mp-weixin
```

### 微信开发者工具
1. 打开微信开发者工具
2. 导入 `dist/build/mp-weixin` 目录
3. 即可预览和调试小程序
